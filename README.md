 **API Security Risk Analysis**
 
 **Objective**

To analyze a public API for potential security risks, assess authentication controls, identify data exposure issues, and provide security recommendations.

 **Tools Used**

* Postman
* Public Demo API
* Canva
* GitHub

 **Key Findings**

* Public Data Exposure
* Missing Authentication Controls
* Multiple User Records Exposure

 **Outcome**

The assessment identified security risks related to data accessibility and authentication. Recommendations were provided to improve API security and reduce potential threats.
